# clock
A web site for curious people who want to know the time.

## Usage
```bash
$ mongod
$ npm install
$ npm run seed
$ npm start
```

## Uses
* HTML5
* CSS3
* Modern JavaScript (ES2015+)
* Moment
* Moment Timezone
* MongoDB
* Express
* Node.js
